Install instructions
--------------------

- Install as usual:

    $ ./configure
    $ make
    # make install

- To Uninstall:

    # make uninstall

### Or for build a Arch Linux packgage:

- From git:

    $ make pacman

- Or from local directory:

    $ make pacman-local

- And then install:

    # pacman -U retrosmart-openbox-*.pkg.xz
